# Cafe Kiosk
C#으로 구현한 카페 키오스크입니다. WinForm을 사용하여 윈도우 애플리케이션을 개발했습니다. MySQL을 연동하여 메뉴와 재고를 관리할 수 있습니다. 

<br>


### 기간
